# 2PT (legacy)

Original Two-Phase Thermodynamics (2PT) reference implementation (v1.4): the
solid + gas hard-sphere density-of-states partition for computing the absolute
entropy and free energy of liquids from a single equilibrium MD trajectory.
Superseded by [py-xPT](../py-xPT); retained here for reference.

**Software archive (Zenodo):** https://doi.org/10.5281/zenodo.7731073

Includes the `2pt.v1.4` source, the user guide (`2pt_user_guide.pdf`), and a
LAMMPS example.
